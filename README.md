## Featured Build

<p align="center">
  <a href="https://github.com/yulong-me/OpenCouncil">
    <img src="https://github-readme-stats.vercel.app/api/pin/?username=yulong-me&repo=OpenCouncil&theme=github_dark&hide_border=true&bg_color=0d1117&title_color=84cc16&text_color=c9d1d9&icon_color=22c55e" alt="OpenCouncil repository card" />

OpenCouncil is a local AI expert collaboration workbench. It lets users create discussion rooms, choose experts, route each message to a target expert, and let experts continue the conversation through `@mention` collaboration.

**Core ideas:** multi-agent rooms, explicit routing, shared context, scenes, providers, workspaces, report generation, local SQLite persistence.
